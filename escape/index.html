脱出への一歩

短時間で遊べる脱出ゲームのアーカイブ。気になるシーンへ飛び込もう。

  • ワンルーム脱出（#1）

    身近な部屋に隠された小さな違和感を追って、最後の鍵を探す。

    所要: 5-7分 難易度: Easy
    ▶ Start

  • 電車内脱出（#2）

    静かな車内に残された手がかりを繋ぎ、閉ざされた扉を開く。

    所要: 6-8分 難易度: Normal
    ▶ Start

  • 図書室脱出（#3）

    静かな書架の間で手がかりを集め、出口を開く。

    所要: 6-8分 難易度: Normal
    ▶ Start

  • 雨音の記録室（#4）

    引き出しの記録を整え、止まった時計を進めて脱出する。

    所要: 12-18分 難易度: Normal
    ▶ Start
